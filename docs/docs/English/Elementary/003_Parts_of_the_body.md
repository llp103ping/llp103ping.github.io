# Head and face
+ head -> brain
+ hair black hairbrush
+ eye watch eyeglasses
+ ear listen earring
+ nose breathe smell
+ mouth eat 
  + tooth/teeth lip **tongue**
  + toothbrush lipstick 
+ neck turn necklace
 <!-- more -->


# Arm and leg
+ shoulder
+ hand 
  + finger ring 
  + **thumb**
  + **nail**
+ leg
  + knee
  + foot/feet football **socks** shose
  + toe


# Rest of body 
+ **chest** -> heart
+ **side** 
+ **stomach** 
+ **back**
+ **waist** 
+ **hip**
+ skin 
+ blood


# Common mistakes
+ one foot -> two feet 
+ one tooth -> two teeth
+ in my(~~the~~) hand/leg...
